# temperature_converter

This is a rewrite of my Python fastHTML app.

It is a simple temperature converter with a clean, minimal interface.

## What it does

- Converts Celsius, Fahrenheit, and Kelvin using single input
- Uses a light and dark theme based on your system settings
- Works on desktop, tablet, and mobile

## Files

- `public/index.html` - page structure
- `public/style.css` - page styling
- `public/temperature.js` - conversion logic
